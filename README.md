# EventVault – Event Management & Reward System

##  Project Overview

**EventVault** is a web based Event Management System designed to help users discover, register, host, and manage events easily. The system also includes a **reward and penalty points mechanism** to encourage responsible participation and active engagement in events.

---

##  Key Features

###  User Management

* User registration and secure login system
* Personal dashboard with profile details
* Track event history and participation

---

###  Event Management

* Browse available events
* Register for events online
* Create and host new events
* View upcoming and past events

---

###  Reward Points System

Users earn points for positive participation:

* +10 points for attending an event
* +15 points for active participation or volunteering
* +25 points for winning an event

---

###  Penalty (Deduction) System

Points are deducted for irresponsible behavior:

* –10 points for not attending after registration
* –5 points for late check-in
* –8 points for last-minute cancellation

---

###  Dashboard & Analytics

* Total events joined
* Points earned
* Upcoming events tracking
* Reward history display

---

##  Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Design:** Responsive UI with modern styling
* **Storage:** Local storage (for demo functionality)

---

##  Objectives

* Simplify event organization and participation
* Encourage user engagement using a reward system
* Reduce manual event management work
* Provide a centralized platform for events

---

##  Developed For

Academic Mini Project – Event Management System with Reward Tracking

---

##  Conclusion

EventVault provides a complete solution for managing events efficiently while motivating users through a structured reward and penalty points system.
